This repository contains babashka scripts.

**Usage Instructions:** To use them, just download the file and place it in `~/.local/bin` folder. You need babashka installed for this to work. Make sure this folder is on PATH, which is the default on most distros. Then make it executable, by doing: `chmod +x ~/.local/bin/name_of_script_file`.

Currently this only includes the babls script.
