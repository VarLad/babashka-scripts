This repository contains babashka scripts.

**Usage Instructions:**
1. To use them, just download the file and place it in `~/.local/bin` folder.
2. Then make it executable, by doing: `chmod +x ~/.local/bin/name_of_script_file`.

**Note**: Make sure the `~/.local/bin` folder is on PATH, which is the default on most distros. You need babashka installed for this to work.

Currently this only includes the babls script.
